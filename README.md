# Metrics-of-MyHome-Online-Platform
Summary of special metrics for MyHome Online Platform.


# Business Objectives
Analyze the number of active users (Builders, PrivateSellers,Broker) for the last twelve months and daily for the current month.
Analyze the movement of listed Buildings along with their apartments and Districts with their TownHoses.
Show the summary of real estate (Apartments, Houses etc.) listed for sale or rent by Brokers and PrivateSellers for the last twelve months and current month.
Summary of registered Stores in the online platform for the last twelve months and current month.
Analyze the revenue per month for the agreements signed vi online platform.
Present the number of total listed real estate via the platform.
Summary of the PromoCodes provided per store for the last twelve months and the current.
Analyze cumulative and monthly Net Promoter Score (NPS) and Customer Experience Index (CEI) for the last twelve months and the current month.


# Data Model
Dim Tables : Loan_Application_Dim_By_date
Fact Tables : Users, Builders, Brokers, PrivateSellers, Apartments, Stores, Promocodes, Surveys


# Tools Used
Microsoft SQL Server
Power BI
